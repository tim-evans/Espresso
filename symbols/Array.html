(Array) flatten(level)

Returns a new array that's a one-dimensional flattening of this array (recursively). That is, for every element that's an array, extract its elements into the new array. If the optional level arguments determines the level of recursion to flatten.

Parameters:
(Number) level Optional
The maximum level of recursion.
Returns:
(Array) The flattened array.
var arr = [1, [2, [3, [4, [5]]]]];
alert(arr.flatten());
// => [1, 2, 3, 4, 5]

alert(arr.flatten(2));
// => [1, 2, 3, [4, [5]]];

(Number) remove(from, to)

Removes the value from the array.

Parameters:
(Number) from
The position to begin removing values from.
(Number) to Optional
The position to remove values to.
Returns:
(Number) The length of the array.
var breakfast = ["banana", "waffles", "bacon", "coffee"];
breakfast.remove(0);

alert(breakfast);
// => ["waffles", "bacon", "coffee"]

breakfast.unshift("sausages", "pancakes")
alert(breakfast);
// => ["sausages", "pancakes", "waffles", "bacon", "coffee"]

breakfast.remove(1, 2);
alert(breakfast);
// => ["sausages", "bacon", "coffee"]
